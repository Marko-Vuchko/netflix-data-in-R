# Netflix Data Analysis Project

## Overview
This project analyzes Netflix content data using R programming language and various data science techniques. The analysis explores patterns in Netflix's movie and TV show catalog, including content types, release years, genres, and other relevant metrics.

## Dataset
The analysis uses several CSV files containing Netflix content information:
- `titles.csv` - Main dataset with movie/show details including:
  - Title ID
  - Title name
  - Content type (MOVIE/SHOW)
  - Description
  - Release year
  - Age certification
  - Runtime
  - Seasons (for shows)
- `certifications.csv`
- `countries.csv` 
- `genres.csv`
- `ratings.csv`

## Tools & Technologies
- R Programming Language
- RStudio
- Key R packages used:
  - Data manipulation 
  - Data visualization
  - Statistical analysis
